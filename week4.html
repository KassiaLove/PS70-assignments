Week 4: Microcontroller Progamming

This week's assignment is to code an Arduino to do something. I was interested to see what I can do with LEDs's so here we go. So first I learned how to use RGB LEDs. I was interested in the fact that you can have multiple colors with just one LED bulb. I watched this Youtube video to get started: https://www.youtube.com/watch?v=IPOHARgRLE0. This video taught me the basics of the RGB LED, how to put it on the breadboard, and how to go about coding it in Arduino. The one thing that was wrong with this video though, was that its breadboard diagram was wrong so it got me stuck for some time. It had the orientation of the resistors wrong so when I uploaded the code, the LED only was showing the color red. But after playing around with the setup and doing some looking around on the internet, I found the way to put the resistors on correctly so that it shows all 3 colors. After this, I got to thinking of how to expand this into a project for this week's assignment.

Materials used:

  • Breadboard
  • Wires
  • ItsyBitsy
  • RGB Led
  • 100 kΩ Resistors
  • Wires

Next, I learned how to make a sequence of LEDs turn on and off in a sequence using this video: https://www.youtube.com/watch?v=e1FVSpkw6q4. At first, it didn't seem that it was working, but then realized that I was using too strong of a resistor. I was using the 100 k&Omega resistors when it seems that the regular LED only needs 1k&Omega to be visible. So after that, I got a cool light-up sequence of LEDs.

Code: 


int LED1 = 13;
int LED2 = 12;
int LED3 = 11;

void setup() {
  // put your setup code here, to run once:
  pinMode(LED1, OUTPUT);
  pinMode(LED2, OUTPUT);
  pinMode(LED3, OUTPUT);
}

void loop() {
  // put your main code here, to run repeatedly:
  digitalWrite(LED1, HIGH); //turn on LED1
  digitalWrite(LED2, HIGH); //turn on LED2
  digitalWrite(LED3, HIGH); //turn on LED3
  delay(1000);               //wait for 1000ms
  digitalWrite(LED1, LOW); //turn off LED1
  delay(300);               //wait for 300ms
  digitalWrite(LED2, LOW); //turn off LED2
  delay(300);               //wait for 300ms
  digitalWrite(LED3, LOW); //turn off LED3
  delay(300);               //wait 300ms before running again
}

After this, I started to experiment with putting multiple RGB LEDs on a breadboard and realized that I can use the same pins on the Itsy Bitsy to make a parallel circuit. So to add more colors that are shown by the RGB LEDs I looked up a couple more color's RGB values and added them to the code to get the whole rainbow.

int RED = 13;

  digitalWrite(GREEN, HIGH);
  digitalWrite(BLUE, HIGH);
  digitalWrite(RED, HIGH);
}
